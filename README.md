# eyebolt.github.io
Eyebolt Main Site

# Setup

Install Jekyll & Dependencies:
```
gem install jekyll
gem install jekyll-paginate
gem install jekyll-gist
gem install redcarpet
gem install pygments.rb
```
Process JS source:
```
npm install -g webpack
npm install
```
# Running

Serve locally:
```
jekyll serve
```
